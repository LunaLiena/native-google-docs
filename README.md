# Соберите приложение

## bash
1. cd src-tauri
2. cargo build --release
3. Найдите EXE файл

4. Перейдите в target/release/

5. Запустите app.exe

6. Создайте ярлык на рабочем столе

7. Правой кнопкой на app.exe → "Создать ярлык"

8. Перетащите ярлык на рабочий стол

9. Переименуйте в "Google Docs"
